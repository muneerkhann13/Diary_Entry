# Diary_Entry
```
Install xampp 


databse name diary_entry
tables 
1)user - name,email,password(sha1)
2)entry - user,date,title,body


place complete folder in htdocs 
go to browser and type localhost/diary-entry
you will be redirected to login.php
and you can sign up there and login 
then make your entry and record memories
```
